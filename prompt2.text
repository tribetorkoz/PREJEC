### PIPELINE COMPONENTS (in this exact order):
1. Transport: LiveKit (for WebRTC) or WebsocketServerTransport
2. STT: DeepgramSTTService
   - model: nova-2
   - language: multi (auto-detect Arabic/English/French)
   - interim_results: True
3. LLM: AnthropicLLMService
   - model: claude-sonnet-4-6
   - max_tokens: 1024
   - system: load from system_prompt.py
4. TTS: ElevenLabsTTSService
   - model_id: eleven_multilingual_v2
   - voice_id: from agent config in database
5. VAD: SileroVADAnalyzer
   - start_secs: 0.2
   - stop_secs: 0.8

### PIPELINE FLOW:
transport.input() -> STT -> LLM -> TTS -> transport.output()

## FILE: backend/agents/system_prompt.py
Create a dynamic system prompt builder:

def build_system_prompt(agent_config, customer_context):
    return f'''
    You are {agent_config.name}, voice assistant for {agent_config.company}.
    
    CUSTOMER CONTEXT:
    {customer_context}
    
    RULES:
    - ALWAYS respond in the same language the customer speaks
    - NEVER speak more than 40 words at once
    - ALWAYS confirm appointments before saving
    - ALWAYS use customer name when known
    - NEVER say you are an AI unless directly asked
    - If asked directly: say you are a digital assistant
    '''
