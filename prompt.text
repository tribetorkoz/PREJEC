# VOICECORE — PROJECT SETUP & FULL ARCHITECTURE

You are an expert full-stack developer. Build a production-ready
B2B Voice AI Agent Platform called VoiceCore.

## PROJECT OVERVIEW
A SaaS platform where businesses subscribe to get an AI voice agent
that answers their customer calls 24/7 on phone and WhatsApp.

## TECH STACK — USE EXACTLY THESE:
- Backend: Python 3.11 + FastAPI
- Voice Engine: Pipecat (pip install pipecat-ai)
- STT (Speech-to-Text): Deepgram
- TTS (Text-to-Speech): ElevenLabs
- LLM: Anthropic Claude Sonnet (claude-sonnet-4-6)
- Phone: Twilio (calls + WhatsApp)
- Real-time: LiveKit WebRTC
- Database: PostgreSQL via Supabase
- Cache/Queue: Redis (Upstash)
- Frontend: Next.js 14 (App Router) + TypeScript
- Styling: Tailwind CSS + shadcn/ui
- Payments: Stripe
- Deploy: Docker + Railway

## DATABASE MODELS — CREATE ALL OF THESE:

// Company (tenant)
Company: id, name, email, plan, stripe_customer_id,
         phone_number, whatsapp_number, created_at

// Voice Agent config per company
Agent: id, company_id, name, language, voice_id,
       system_prompt, is_active, created_at

// Every call record
Call: id, company_id, caller_phone, duration_seconds,
      transcript, sentiment, outcome, created_at

// Customer memory
Customer: id, company_id, phone, name, notes,
          last_call_at, total_calls
